# 1. Przewodowe

Protokoły przewodowe charakteryzują się duża przpustowością, niezawodnością, oraz bezpieczeństwem

**Przemysł:**

EtherCAT
IOLink
Modbus
Profibus
Profinet
ASI

**Automatyka budynkowa:**

KNX
Modus
DALI

# 2. Bezprzewodowe

Protokoły bezprzewodowe mają mniejszą przepustowość są mniej bezpieczne, a jakosć sygnału zależna jest od rodzaju pomieszczenia i przeszkód

**Przemysł:**

dyskusyjna technolaogia z racji na wysokie ryzyko utraty danych w procesie co w przemyśle może być kluczowe

WirelessHART
WiFi
LoRa WAN

**Automatyka budynkowa:**

Zigbee
WiFi
KNX RF
