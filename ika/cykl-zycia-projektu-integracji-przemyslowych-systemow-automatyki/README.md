# 1. Planowanie

- zebranie zakresu projektu
- identyfikacja zespołu
- ustalenie budżetu
- ustalenie harmonogramu

# 2. Projektowanie inżynieria

- wybór sprzętu i oprogramowania
- projektowanie strategii sterowania
- tworzenie schematów okablowania i sieci

# 3. Zamówienia

- zakup sprzętu, komponentów i oprogramowania
- wybór dostawców (może przetarg)
- negocjacje umów
- zarządzanie dostawą

# 4. Instalacja i uruchomienie

- instalacja i uruchomienie.
- konfiguracja/Kalibracja
- przeprowadzenie testów
- testy bezpieczeństwa

# 5. Szkolenie i dokumentacja

- opracowanie procedur operacyjnych
- podręczników konserwacji
- stworzenie dokumentacji
- przeprowadzenie sesji szkoleniowej docelowym pracownikom (panu Januszowi utrzymaniowcowi ruchu na zakładzie)

# 6. Obsługa i konserwacja

- monitorowanie wydajności
- serwis
- modyfikacje w razie konieczności

# 7. Wycofanie z eksploatacji

- wyłączenie (jak giga piec to też żeby dziwnie materiał nie zastygł np. w środku)
- usunięcie sprzętu, materiałów i utylizację
