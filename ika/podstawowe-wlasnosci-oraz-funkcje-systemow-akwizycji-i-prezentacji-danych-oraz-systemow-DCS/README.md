# Systemy akwizycji i prezentacji danych (SCADA - Supervisory Control And Data Acquisition)

- Monitorowanie procesu w czasie rzeczywistym

- Zbieranie danych z czujników, przetworników, sterowników PLC

- Przechowywanie danych w bazach lub rejestrach historycznych

- Wizualizacja – panele operatorskie, wykresy, alarmy

- Dostęp zdalny – z poziomu komputerów lub urządzeń mobilnych

- Sterowanie w ograniczony sposób - załączanie pomp, otwarcie zaworu itp.)

## Funkcje:

1. Akwizycja danych
  - zbieranie wartości pomiarowych z rozproszonych urządzeń

2. Przetwarzanie danych
  - filtracja, skalowanie, obliczenia pomocnicze

3. Prezentacja danych
  - wykresy trendów, panele operatorskie, raporty

4. Alarmowanie i powiadamianie
  - wykrywanie stanów niebezpiecznych lub przekroczeń parametrów

5. Architektura rozproszona
  - możliwość integracji wielu urządzeń i lokalizacji

# Systemy DCS (Distributed Control System)

Rozproszone sterowanie procesem – wiele sterowników lokalnych połączonych siecią

- Centralne nadzorowanie – operator ma pełny widok procesu
- Redundancja i niezawodność – kluczowe w procesach ciągłych i krytycznych
- Modularność – łatwe dodawanie nowych modułów sterowania i wejść/wyjść

## Funkcje:

1. Sterowanie procesem
- regulacja automatyczna i sekwencyjna
2. Integracja urządzeń
- PLC, czujniki, aktuatory, systemy pomiarowe
3. Monitoring i wizualizacja
- HMI (Human Machine Interface), panele operatorskie, alarmy
4. Akwizycja i archiwizacja danych
- zapis trendów, raporty, analiza historii procesu
5. Zarządzanie produkcją i diagnostyka
- optymalizacja wydajności, wczesne wykrywanie usterek
